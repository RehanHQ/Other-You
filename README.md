# OtherYou

Meet the versions of yourself that never existed.

An AI-powered multiverse simulator that generates alternate life paths based on your real choices.

## Setup

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env` file:

```
GROQ_API_KEY=your_groq_api_key_here
```

## Deploy to Vercel

1. Push to GitHub
2. Import in Vercel
3. Add `GROQ_API_KEY` environment variable
4. Deploy

## Tech Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- Groq AI API
- Vercel Serverless Functions
